# Identifying Farmland Characteristics

Machine learning analysis of the Agriculture Vision 2021 dataset to identify farmland characteristics including nutrient deficiencies, water presence, and weed coverage.

## Project Structure

- `src/` - Analysis code and Jupyter notebooks
- `paper/` - LaTeX source files and compiled PDF (`Writeup.pdf`)

## Dataset

This project uses the Agriculture Vision 2021 dataset. Download from [Google Drive](https://drive.google.com/open?id=1kXLq7ronLoS_m_iEVoOYx7mgEoMWecB6&usp=drive_fs) when working locally. For JupyterHub server downloads, refer to the assignment handout.

**Note:** Data files should not be committed to the repository.

## Usage

See `src/README.md` for detailed instructions on running the analysis notebooks.
